# Webpack Starter Kit
- This is a very simple starter kit with webpack
- It uses Webpack-dev-server so the bundle files will only be present during development
- Start Developing
  ~~~
    $ npm start -s
  ~~~
- Component Docs
  ~~~
    $ npm run storybook -s
  ~~~
